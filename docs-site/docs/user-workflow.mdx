# User Workflow

The typical `Pheno-Ranker` workflow has four stages: prepare input data, run a comparison, inspect or visualize the outputs, and optionally use companion utilities. Start with your data format, choose the comparison mode, then follow the output to the analysis or utility page that matches your task.
