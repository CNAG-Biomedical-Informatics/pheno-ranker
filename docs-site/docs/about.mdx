**Pheno-Ranker** is developed at [CNAG](https://cnag.eu), Barcelona, Spain, with contributions from collaborators who helped shape the command-line interface, web interface, utilities, and documentation.

<div className="about-hero">
  <p>
    The project brings together phenotype similarity scoring, Beacon v2 and
    Phenopackets data models, graph export, barcode utilities, and reusable
    command-line workflows for comparing patient or cohort profiles.
  </p>
</div>
