## Project Status

The documentation follows the current GitHub version of `Pheno-Ranker`; the latest CPAN release can differ between releases. Consult the [Changes file](https://github.com/CNAG-Biomedical-Informatics/pheno-ranker/blob/main/Changes) for version-specific updates. The CLI is the primary interface, while the legacy Web App remains supported until 2027.
