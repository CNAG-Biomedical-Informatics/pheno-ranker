:::tip[Primary interface]
The `pheno-ranker` command-line interface is the main entry point for reproducible file-based analyses, configuration files, large cohorts, and result exports.

The Web App is available for interactive use, while the Perl module supports programmatic integration.
:::

## Typical Analyses

Most users come to `Pheno-Ranker` for one of these tasks:

- compute all-vs-all Hamming distances or Jaccard similarities within or across cohorts
- rank one or more target records against a reference cohort
- compare Beacon v2 Models, Phenopackets v2, or configured generic JSON/YAML records
- prepare CSV data with `csv2pheno-ranker`
- export dense or sparse matrices, ranking statistics, coverage data, or thresholded graphs
